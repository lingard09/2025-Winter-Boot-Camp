B04. 5명의 국영수 총점 평균 구하기

학생 5명의 국어, 영어, 수학 점수를 각각 입력받은 후에, 각각의 총점과 평균 점수를 출력하라.

총점 = 국어점수 + 영어점수 + 수학점수,
평균 = 총점 / 3.0 으로 계산하라.

변수는 다음과 같이 사용하라.

int jumsu[5][3];  // 5명의 3과목 점수를 저장하는 2차원 배열
int sum[5];  // 학생별 총점
float average[5];  // 학생별 평균

입력 형식

5줄에 걸쳐 정수 3개 (각각 국어, 영어, 수학 점수)

출력 형식

5줄에 걸쳐 번호, 총점, 평균(소숫점 1자리)

예제

예제 입력 1

95 85 75
87 79 88
95 85 75
65 77 79
77 88 99


예제 출력 1

1) 255 85.0
2) 254 84.7
3) 255 85.0
4) 221 73.7
5) 264 88.0

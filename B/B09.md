B09. 여러 명의 점수 분석하기 (구조체 포인터 배열 버전)

다음과 같은 구조체를 사용하여, 여러 명의 국어, 영어, 수학 점수를 각각 입력받아 저장한 후에, 다음 항목들을 계산하여 출력하라.

각 학생별 총점, 평균점수, 평균에 따른 등급
각 과목별 이름, 총점, 평균점수

등급은 다음과 같은 기준으로 판정하라.

평균 90이상 : A
평균 80이상 ~ 90미만 : B
평균 70이상 ~ 80미만 : C
평균 60이상 ~ 70미만 : D
평균 60미만 : F

단, 다음과 같은 구조체를 사용하라.

struct st_jumsu{
char name[20]; // 이름 (공백 없이 영어 알파벳으로 구성)
int jumsu[3]; // 3과목 점수 (국, 영, 수)
int sum;	// 총점
float avg;	// 평균
char grade;	// 평균에 대한 등급
};

반드시 다음과 같은 함수를 제작하여 사용하라.

char getGrade(float score);
//파라미터: 평균 점수(score)
//리턴값: 해당 평균 점수의 등급 ('A','B','C','D','F')

void makeJumsu(struct st_jumsu* p);
//파라미터: 점수 데이터의 포인터(p)
//리턴값: 없음
//수행내용: 해당 학생의 점수 데이터 구조체 내의 과목 점수로 총점, 평균, 등급을 넣는다.

void printJumsu(struct st_jumsu* p);
//파라미터: 점수 데이터의 포인터(p)
//리턴값: 없음
//수행내용: 해당 학생의 점수 데이터 구조체 내의 값을 출력한다.

int getClassSum(struct st_jumsu* data[], int size, int index);
//파라미터: 점수 데이터의 포인터 배열(data), 점수 데이터 개수, 과목번호(0~2)
//리턴값: 과목번호에 해당하는 점수의 총 합계
//수행내용: 모든 점수 데이터의 특정 과목번호의 점수 합계를 구한다.

변수는 다음과 같이 사용하라.

struct st_jumsu* jdata[20]; // 최대 20명의 점수 데이터
int sum_class[3]; // 과목별 총점
float average_class[3]; // 과목별 평균
char class_name[3][20] = {"Korean", "English", "Math"};

입력 형식

첫 줄에 점수를 입력할 인원 수, 둘째 줄부터 이름과 점수 3개 (각각 국어, 영어, 수학 점수)

출력 형식

인원 수만큼 번호, 이름, 총점, 평균(소숫점 1자리), 등급, 그 다음 3줄에 걸쳐 과목명, 과목 총점, 과목 평균

예제

예제 입력 1

5
James 95 86 75
David 88 59 61
Sarah 95 95 95
Mathew 65 77 79
Brian 57 58 49

James 256 85.3 B
David 208 69.3 D

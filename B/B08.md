B08. 3과목 점수의 평균 구하고, 등급 판정하기 (구조체 버전)

국어, 영어, 수학 점수를 입력받은 후에, 3과목의 평균 점수를 출력하고, 이 점수의 등급을 판정하여 출력하라.

총점 = 국어점수 + 영어점수 + 수학점수
평균 = 총점 / 3.0 으로 계산하라.

등급은 다음과 같은 기준으로 판정하라.

평균 90이상 : A
평균 80이상 ~ 90미만 : B
평균 70이상 ~ 80미만 : C
평균 60이상 ~ 70미만 : D
평균 60미만 : F

단, 다음과 같은 구조체를 사용하라.

struct st_jumsu{
char name[20]; // 이름 (공백 없이 영어 알파벳으로 구성)
int jumsu[3]; // 3과목 점수 (국, 영, 수)
int sum;	// 총점
float avg;	// 평균
char grade;	// 평균에 대한 등급
};

반드시 다음과 같은 함수를 제작하여 사용하라.

char getGrade(float score);
//파라미터 : 평균 점수(score)
//리턴값 : 해당 평균 점수의 등급 ('A','B','C','D','F')

변수는 다음과 같이 사용하라.

struct st_jumsu one; // one의 점수 데이터

입력 형식

이름과 정수 3개 (각각 국어, 영어, 수학 점수)

출력 형식

이름, 총점, 평균(소숫점 1자리)과 등급

예제

예제 입력 1

James 95 85 88


예제 출력 1

James 268 89.3 B


예제 입력 2

David 95 99 94


예제 출력 2

David 288 96.0 A

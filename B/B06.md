B06. 5명의 점수 분석하기 (함수 버전)

5명의 국어, 영어, 수학 점수를 각각 입력받아 저장한 후에, 다음 항목들을 계산하여 출력하라.

각 학생별 총점과 평균점수, 평균에 따른 등급
각 과목별 총점과 평균 점수

등급은 다음과 같은 기준으로 판정하라.

평균 90이상 : A
평균 80이상 ~ 90미만 : B
평균 70이상 ~ 80미만 : C
평균 60이상 ~ 70미만 : D
평균 60미만 : F

반드시 다음과 같은 함수를 제작하여 사용하라.

int studentSum(int s[5][3], int number);
//파라미터 : 점수배열(s)과 학생번호(0,1,2,3,4)
//리턴값 : 해당 학생의 총점

char getGrade(float score);
//파라미터 : 평균 점수(score)
//리턴값 : 해당 평균 점수의 등급 ('A','B','C','D','F')

int classSum(int s[5][3], int number);
//파라미터 : 점수배열(s)과 과목번호(0,1,2)
//리턴값 : 해당 과목의 총점

변수는 다음과 같이 사용하라.

int jumsu[5][3]; // 5명의 3과목 점수를 저장하고 있는 2차원 배열
int sum_student[5]; // 학생별 총점
float average_student[5];  // 학생별 평균
char grade[5]; // 학생별 등급

int sum_class[3]; // 과목별 총점
float average_class[3]; // 과목별 평균
char class_name[3][20] = {"Korean", "English", "Math"};

입력 형식

5줄에 걸쳐 정수 3개 (각각 국어, 영어, 수학 점수)

출력 형식

5줄에 걸쳐 번호, 총점, 평균(소숫점 1자리), 등급, 그 다음 3줄에 걸쳐 과목명, 과목 총점, 과목 평균

예제

예제 입력 1

95 86 75
88 59 61
95 95 95
65 77 79
57 58 49


예제 출력 1

1) 256 85.3 B
2) 208 69.3 D
3) 285 95.0 A
4) 221 73.7 C
5) 164 54.7 F
Korean 400 80.0

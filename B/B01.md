B01. 3과목 점수의 총점, 평균 구하기

국어, 영어, 수학 점수를 입력받아 3과목의 점수의 총점과 평균을 계산하여 출력하라.

총점 = 국어점수 + 영어점수 + 수학점수
평균 = 총점 / 3.0 으로 계산하라.

변수는 다음과 같이 사용하라.

int kor, eng, math; // 국어점수, 영어점수, 수학점수 
int total; // 총점
float average; // 평균점수


입력 형식

정수 3개 (각각 국어, 영어, 수학 점수)

출력 형식

총점과 평균(소숫점 1자리)

예제

예제 입력 1

85 95 80


예제 출력 1

260 86.7

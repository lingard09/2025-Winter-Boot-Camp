A06. 10명 중 누가 비만인가? (함수 버전)

10명의 신장(cm단위)과 체중(kg단위)를 입력받은 후, 이 사람들 중 비만인 사람의 순서와 총 몇 명이 비만인지 출력하라.

단, A05문제에서 제작한 bmiLevel() 함수를 사용하여 비만 여부를 판정하라. (비만등급이 3 이상인 경우 비만임)

int bmiLevel(int height, int weight);
//파라미터: height: 신장(cm), weight: 체중(kg)
//수행내용: 비만도 계산 후 비만등급을 판정한다.
//리턴값: 비만등급 (0~4): 저체중 0, 정상체중 1, 과체중 2, 경도비만 3, 고도비만 4

비만도 수치 = 체중(kg) / (신장(m)의 제곱) 으로 계산
비만등급 (0~4):비만도 수치(bmi)에 따른 비만등급 판정 기준은 다음과 같다.

bmi < 18.5: 0 (Underweight),
18.5 <= bmi < 23: 1 (Normal weight),
23 <= bmi < 25: 2 (Overweight),
25 <= bmi < 30: 3 (Mild obesity),
30 <= bmi: 4 (Severe obesity)

변수는 다음과 같이 사용하라.

int height[10], weight[10]; // 10명의 신장(cm), 체중(kg)
int bmi_level[10]; // 10명의 비만등급(0~4)
int count; // 비만인 사람의 숫자
그 외의 변수는 필요 시 임의로 사용하라

입력 형식

10줄에 걸쳐 2개의 정수 - 신장(cm단위)과 체중(kg단위)

출력 형식

첫 줄에는 비만인 사람의 순서, 두번째 줄에는 총 비만인 사람 수

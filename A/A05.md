A05. 비만 등급을 판정하라 (함수 버전)

신장(cm단위)과 체중(kg단위)를 입력받은 후, 비만도(BMI)에 따른 비만등급을 출력하라.


    비만도 수치 = 체중(kg) / (신장(m)의 제곱) 으로 계산한다.

    단, bmiLevel() 함수를 제작하여 이를 사용해 비만등급을 판정하도록 하라.


int bmiLevel(int height, int weight);
//파라미터: height: 신장(cm), weight: 체중(kg)
//수행내용: 비만도 계산 후 비만등급을 판정한다.
//리턴값: 비만등급 (0~4): 0 (Underweight), 1 (Normal weight), 2 (Overweight), 3 (Mild obesity), 4 (Severe obesity)


비만등급 (0~4):비만도 수치(bmi)에 따른 비만등급 판정 기준은 다음과 같다.

bmi < 18.5: 0 (Underweight),
18.5 <= bmi < 23: 1 (Normal weight),
23 <= bmi < 25: 2 (Overweight),
25 <= bmi < 30: 3 (Mild obesity),
30 <= bmi: 4 (Severe obesity)

main()에서 출력을 위해 다음 문자열 배열을 사용하라.

char bmi_string[5][20] = {"Underweight","Normal weight","Overweight","Mild obesity","Severe obesity"};


입력 형식

2개의 정수 - 신장(cm단위)과 체중(kg단위)

출력 형식

비만 등급 문자열과 등급 수치(0~4)

예제

예제 입력 1

176 80


예제 출력 1

Mild obesity (3)


예제 입력 2

168 54


예제 출력 2

Normal weight (1)


예제 입력 3

170 45


예제 출력 3

Underweight (0)


예제 입력 4

176 98


예제 출력 4

Severe obesity (4)

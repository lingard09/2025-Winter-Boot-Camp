A02. 비만 판정

신장(cm단위)과 체중(kg단위)를 입력받은 후, 비만도 수치와 비만 여부판정 결과를 출력하라.

비만도 수치 = 체중(kg) / (신장(m)의 제곱) 으로 계산한다.
단, 비만 여부판정 기준은 비만도 수치가 25 이상인 경우에 “Overweight”, 그 외의 경우에 “Normal” 이라고 판단한다.
이 때, 신장은 미터 단위로 환산해야 함을 유의하라.

변수는 다음과 같이 사용하라.

int height, weight; // 신장(cm), 체중(kg)
float bmi; // 비만도 수치

입력 형식

2개의 정수 - 신장(cm단위)과 체중(kg단위)

출력 형식

비만도 수치(소수점 1자리)와 판정 결과 문자열

예제

예제 입력 1

165 70


예제 출력 1

25.7 Overweight


예제 입력 2

185 70


예제 출력 2

20.5 Normal

D02. 영화정보 관리 프로그램 (ver.2)

다음과 같은 영화 정보를 최대 50개까지 입력받아 영화목록, 쟝르별 영화 개수, 상영시간이 가장 긴 영화, 관객 수가 가장 많은 영화를 출력하는 프로그램을 제작하라.

영화제목 (공백이 포함된 문자열, 최대 100글자)
개봉연도 (숫자)
쟝르번호 (숫자, 0~7)
상영시간 (숫자, 분 단위)
관객수 (숫자)
영화소개글 (공백이 포함된 문자열, 최대 255글자)

단, 영화정보는 다음과 같이 구조체 타입으로 미리 정해져있다.

typedef struct st_movie{
char title[100];
int year, genre, playtime, viewers;
char description[255];
} MOVIE;

쟝르번호는 다음과 같이 미리 정해져있다.

const char* genre_name[] = {
"Sci-Fi",
"Thriller",
"Action",
"Romance",
"Musical",
"Drama",
"Animation",
"War"
};

반드시 다음과 같은 함수를 제작하여 사용하라.

int loadRecords(MOVIE* list[]);

영화 개수 및 그만큼의 영화 정보를 입력받아 목록에 추가한다.


    파라미터 : 목록 포인터 배열

    리턴값 : 읽어온 목록 수


void printMovie(MOVIE* p);

특정 영화 정보를 출력한다.


    파라미터 : 영화정보 구조체 포인터

    리턴값 : 없음


void printReport(MOVIE* list[], int size);

쟝르별 영화 개수, 상영시간이 가장 긴 영화, 관객 수가 가장 많은 영화 정보를 출력한다.


    파라미터 : 목록 포인터 배열, 목록크기

    리턴값 : 없음

    주의사항 : 이 함수에서 printGenreCount(), maxViewers(), maxPlaytime() 함수를 사용한다.


int indexGenre(char* name);

쟝르 이름에 해당하는 인덱스 값을 찾는다.


    파라미터 : 쟝르이름

    리턴값 : 해당 쟝르의 인덱스 값


void printGenreCount(MOVIE* list[], int size);

영화 전체목록에서 쟝르별로 영화의 개수를 세어 출력한다.


    파라미터 : 목록 포인터 배열, 목록크기

    리턴값 : 없음


int maxViewers(MOVIE* list[], int size);

영화 전체목록에서 관객수가 가장 많은 영화의 인덱스를 찾는다.


    파라미터 : 목록 포인터 배열, 목록크기

    리턴값 : 인덱스(0~size-1)


int maxPlaytime(MOVIE* list[], int size);

영화 전체목록에서 상영시간이 가장 긴 영화의 인덱스를 찾는다. (완성코드 제공)


    파라미터 : 목록 포인터 배열, 목록크기

    리턴값 : 인덱스(0~size-1)


main() 함수는 다음과 같이 사용하라.

int main() {
MOVIE* mlist[50];
int count;
count = loadRecords(mlist);
for(int i=0; i<count; i++){
printf("%d) ", i+1);
printMovie(mlist[i]);
}
printReport(mlist, count);
return 0;
}

반드시 loadRecords()에서 구조체 포인터 배열의 요소에 메모리를 할당하는 작업이 필요하다.

입력 형식

첫째 줄에 입력할 영화 개수, 다음 줄부터 3줄 단위로 영화 제목 / 영화 소개글 / 개봉년도 쟝르 상영시간(분) 관객수

출력 형식

영화목록, 쟝르별 영화 개수, 상영시간이 가장 긴 영화, 관객 수가 가장 많은 영화 (아래 출력 참고)

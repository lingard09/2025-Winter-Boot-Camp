D01. 영화정보 관리 프로그램 (ver.1)

다음과 같은 영화 정보를 최대 50개까지 입력받아 영화목록을 출력하는 프로그램을 제작하라.

영화제목 (공백이 포함된 문자열, 최대 100글자)
개봉연도 (숫자)
쟝르번호 (숫자, 0~7)
상영시간 (숫자, 분 단위)
관객수 (숫자)
영화소개글 (공백이 포함된 문자열, 최대 255글자)

단, 영화정보는 다음과 같이 구조체 타입으로 미리 정해져있다.

typedef struct st_movie{
char title[100];
int year, genre, playtime, viewers;
char description[255];
} MOVIE;

쟝르번호는 다음과 같이 미리 정해져있다.

const char* genre_name[] = {
"Sci-Fi",
"Thriller",
"Action",
"Romance",
"Musical",
"Drama",
"Animation",
"War"
};

반드시 다음과 같은 함수를 제작하여 사용하라.

void printMovie(MOVIE* p);

특정 영화 정보를 출력한다.


    파라미터 : 영화정보 구조체 포인터

    리턴값 : 없음


int indexGenre(char* name);

쟝르 이름에 해당하는 인덱스 값을 찾는다.


    파라미터 : 쟝르이름

    리턴값 : 해당 쟝르의 인덱스 값


main() 함수는 다음 코드를 참고하라.

int main() {
MOVIE* list[10];
int count;
char title[100], desc[255], genre[50];
int year, playtime, viewers;

scanf("%d", &count);
for(int i=0; i<count; i++){
	getchar();
	fgets(title, 100, stdin);

	scanf("%d %s %d %d", &year, genre, &playtime, &viewers);
	list[i] = (MOVIE*) malloc(sizeof(MOVIE));

// Your code here!

return 0;


}

입력 형식

첫째 줄에 입력할 영화 개수, 다음 줄부터 3줄 단위로 영화 제목 / 영화 소개글 / 개봉년도 쟝르 상영시간(분) 관객수

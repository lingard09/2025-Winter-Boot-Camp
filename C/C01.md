C01. 외국인 학생 정보 입력받아 출력하기 (구조체 버전)

한 외국인 학생의 정보(이름, 성별, 국적, 태어난 년도)를 입력받아 구조체로 저장한 후, 이 정보를 적절히 출력하라.

국적은 인덱스에 따라 다음과 같이 정한다.


    0 - “KR”

    1 - “US”

    2 - “JP”

    3 - “CN”

    4 - “FR”

    5 - “–” // 다른 국적


성별은 숫자로 관리하며, 다음과 같다.


    0 - “Female”

    1 - “Male”


다음과 같은 구조체를 사용하라.

struct st_person{
char name[20]; 	// Name (single word, no duplicates)
int gender;  	// 0 - Female, 1 - Male
int country;	// Country code (index of COUNTRY_NAME 0~5)
int birthyear; 	// Birthyear
};

반드시 다음과 같은 함수를 제작하여 사용하라.

int getCountry(char* str);
파라미터: 국적을 나타내는 문자열(str)
리턴값: 국적에 해당되는 인덱스 (0~5)
수행내용: 국적 문자열에 해당하는 인덱스를 찾는다.

void printInfo(struct st_person* p);
파라미터: 학생정보 구조체 포인터(p)
리턴값: 없음
수행내용: 해당 학생 정보의 내용을 출력한다.

변수는 다음과 같이 사용하라.

struct st_person* one; // 학생 1명의 데이터
char country[5]; // 국적입력을 위한 문자열

반드시 구조체 포인터 one에 메모리를 할당하는 작업이 필요하다.

입력 형식

이름과 성별인덱스, 국적문자열, 태어난 년도

출력 형식

인적사항 정보 (아래 출력 참고)

예제

예제 입력 1

James 1 KR 2000


예제 출력 1

James (Male, age:25, from KR)


예제 입력 2

Mary 0 FR 2004


예제 출력 2

Mary (Female, age:21, from FR)

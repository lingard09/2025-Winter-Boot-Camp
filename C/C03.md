C03. 외국인 학생 정보 관리 프로그램

여러 명의 외국인 학생 정보(이름, 성별, 국적, 태어난 년도)를 입력받이 구조체 포인터 배열에 저장한 후, 국적에 따라 성별 인원수와 평균 나이를 출력하라.

국적은 인덱스에 따라 다음과 같이 정한다.


    0 - “KR”

    1 - “US”

    2 - “JP”

    3 - “CN”

    4 - “FR”

    5 - “–” // 다른 국적


성별은 숫자로 관리하며, 다음과 같다.


    0 - “Female”

    1 - “Male”


다음과 같은 구조체를 사용하라.

struct st_person{
char name[20]; 	// Name (single word, no duplicates)
int gender;  	// 0 - Female, 1 - Male
int country;	// Country code (index of COUNTRY_NAME 0~5)
int birthyear; 	// Birthyear
};

반드시 다음과 같은 함수를 제작하여 사용하라. 필요한 경우 함수를 추가하여 제작하라.

int getCountry(char* str);
파라미터: 국적을 나타내는 문자열(str)
리턴값: 국적에 해당되는 인덱스 (0~5)
수행내용: 국적 문자열에 해당하는 인덱스를 찾는다.

void printInfo(struct st_person* p);
파라미터: 학생정보 구조체 포인터(p)
리턴값: 없음
수행내용: 해당 학생 정보의 내용을 출력한다.

변수는 다음과 같이 사용하라. 필요한 변수는 추가로 선언하라.

struct st_person* pdata[20]; // 최대 20명의 데이터
char country[5]; // 국적입력을 위한 문자열
int count;  // 인원수

입력 형식

첫 줄에 입력할 인원 수, 둘째 줄부터 이름과 성별인덱스, 국적문자열, 태어난 년도

출력 형식

인원 수만큼 인적사항 정보 (아래 출력 참고), 그 다음 줄에 국적에 따라 성별 인원수와 평균 나이(소숫점 1자리)
